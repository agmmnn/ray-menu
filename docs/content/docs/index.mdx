# Welcome to Ray Menu

Ray Menu is a modern, framework-agnostic radial/pie menu library built with TypeScript. It provides a Web Component for universal use and first-class React bindings.

## Key Features

- **Zero Dependencies** - Pure TypeScript core with no external dependencies
- **Framework Agnostic** - Works anywhere with the Web Component, plus React bindings
- **Infinite Radial Selection** - Sectors extend infinitely outward based on angle
- **Keyboard Navigation** - Full keyboard support for accessibility
- **Nested Submenus** - Support for async-loaded submenus
- **CSS Theming** - Customize with CSS variables

## Quick Start

### Web Component

```html
<script type="module">
  import 'ray-menu/wc';
</script>

<ray-menu>
  <ray-menu-item value="copy">Copy</ray-menu-item>
  <ray-menu-item value="paste">Paste</ray-menu-item>
  <ray-menu-item value="cut">Cut</ray-menu-item>
</ray-menu>
```

### React

```tsx
import { TheHelm, useRadialMenu } from 'ray-menu/react';

const items = [
  { id: 'copy', label: 'Copy' },
  { id: 'paste', label: 'Paste' },
  { id: 'cut', label: 'Cut' },
];

function App() {
  const menu = useRadialMenu({
    items,
    onSelect: (item) => console.log('Selected:', item.label),
  });

  return <TheHelm menu={menu} />;
}
```

## How It Works

Ray Menu uses a radial selection model:

1. **Trigger** - Right-click or long-press to open the menu
2. **Navigate** - Move your cursor/finger in the direction of the item
3. **Select** - Release to select the highlighted item

The menu uses **infinite radial selection** - items are selected based on the angle from center, not the distance. This makes selection fast and intuitive.
