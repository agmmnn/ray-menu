# Installation

Ray Menu can be installed via npm, yarn, pnpm, or bun.

## Package Managers

```bash
# npm
npm install ray-menu

# yarn
yarn add ray-menu

# pnpm
pnpm add ray-menu

# bun
bun add ray-menu
```

## Package Structure

| Entry Point | Description |
|------------|-------------|
| `ray-menu` | Web Component + core logic (zero dependencies) |
| `ray-menu/core` | Core logic only (for custom implementations) |

## CDN Usage

You can use Ray Menu directly from a CDN:

```html
<script type="module">
  import 'https://esm.sh/ray-menu';
</script>

<ray-menu id="menu"></ray-menu>
```

## TypeScript

Ray Menu includes TypeScript definitions out of the box. No additional `@types` package is needed.
